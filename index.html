Projects

Multi Biometric Door Lock System

The Multi-Biometric Door Lock System enhances home security by integrating fingerprint and ID card recognition for robust access control. Using advanced biometric authentication, it ensures only authorized individuals can enter, while remote access capabilities allow homeowners to manage entry via a secure mobile application. This innovative system combines security and convenience for modern residential needs.

Obstacle Avoider Rover

The Obstacle Avoider Rover is an autonomous vehicle designed to navigate its environment by detecting and avoiding obstacles using ultrasonic sensors. Controlled by a microcontroller like Arduino, it processes sensor data to adjust its path and prevent collisions. This project demonstrates practical applications in automated transport and service robots, showcasing integration of robotics and sensor technology.
